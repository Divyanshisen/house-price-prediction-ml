# House Price Prediction Model

## Overview
This project was developed as part of the Bee Skilled Internship Week 2 tasks.

The objective is to predict house prices using Linear Regression and evaluate the model using the R² Score.

## Dataset
Housing Dataset (Housing.csv)

## Features Used
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking

## Machine Learning Algorithm
- Linear Regression

## Steps Performed
1. Data Loading
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training
6. House Price Prediction
7. R² Score Evaluation
8. Actual vs Predicted Visualization

## Libraries Used
- Pandas
- Matplotlib
- Scikit-Learn

## Output
The model predicts house prices based on selected housing features and evaluates performance using the R² Score.
